# Expander

## Usage

See [o-expander](https://github.com/Financial-Times/o-expander) documentation for usage information.

### Sass

This module is _not_ bundled by default.

```scss
@import '~n-ui/expander/main';
```

### JS

This module is _not_ bundled by default.

```js
const expander = require('n-ui/expander');

expander.init();
```
